Performs all the processes to both generate and split up the session key, returning an array of packets to encasulate in the Adapter.

Parameters
mMessage, or the session key to be generated, see SMAdaoter::startSessionKeyExchange(omnetpp::cMessage* msg).
lLength of the session key, does not need to be a multiple of 4 since char('0')s (or int(48)) are padded to reach the desired length.
trustedPartyIdID of the Trusted Third Party to share a session key with.
trustedPartyKeyMissnamed variable, this is just the first piece of the public key - specificially the component generated by the Utility Company.
trustedPartyMusecond peice of the public key, generated by the Trusted Third Party.
Returns
Returns array of messages to be encapsulated and sent to Trusted Third Party. Array is of length (l+(4-(l%4)))/4, or ceiling(l/4) where l is the length of the session key.
